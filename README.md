# Freda Karol — Landing Page (GitHub Pages)

Sito statico pronto per essere pubblicato su **GitHub Pages**.

## Struttura
- `index.html`
- `styles.css`
- `script.js`
- `assets/` (immagini e favicon)

## Pubblicazione su GitHub Pages
1. Crea un repository (es. `freda-karol`).
2. Carica dentro **tutti** i file di questo zip (root del repo).
3. Vai su **Settings → Pages**.
4. In *Build and deployment* seleziona:
   - Source: **Deploy from a branch**
   - Branch: **main** / root
5. Salva. Dopo poco avrai l'URL pubblico.

> Se vuoi collegare i pulsanti “Acquista” a un tuo shop:
> - sostituisci `href="#"` nella sezione `#acquista` con il link corretto.
